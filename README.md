# marxiv
arxiv reader and bibliometric analysis

Example:
```sh
./marxiv.py 2310.20256 | glow -p
```

## Install

Pandoc is required.

```sh
sudo apt install pandoc
pip install -e .
```
For macOS and Windows, see: https://pandoc.org/installing.html
